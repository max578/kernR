# Cross-Fit Propensity Scores

Cross-Fit Propensity Scores

## Usage

``` r
cross_fit_propensity(treatment, covariates, method, n_folds)
```

## Arguments

- treatment:

  Binary vector.

- covariates:

  Numeric matrix.

- method:

  Classification method.

- n_folds:

  Number of folds.

## Value

Vector of out-of-sample propensity scores.
